## Usage

### Standalone (browser)

Open `index.html` in any modern browser. A settings gear (âš™) in the corner opens the in-page control panel.

### Lively Wallpaper

1. Add `index.html` as a new wallpaper source in [Lively Wallpaper](https://github.com/rocksdanister/lively).
2. Use the bundled `LivelyProperties.json` to expose all settings in Lively's property panel (speed, colors, Conway mode, etc.).

## Project Structure

```
index.html              Entry point
style.css               Layout and theme styles
js/
  constants.js          Shared constants (cell size, timing, color palettes)
  state.js              Central mutable game state
  canvas.js             Canvas sizing helpers
  grid.js               Grid utilities (bounds, food placement, occupied-cell map)
  ai.js                 Snake pathfinding / direction logic
  dungeon.js            Conway's Game of Life wall generation
  renderer.js           Canvas draw loop
  game.js               Per-snake game tick, death/respawn, satiety logic
  loop.js               requestAnimationFrame main loop
  ui.js                 Settings panel interactions
  lively.js             Lively Wallpaper property bridge
LivelyProperties.json   Lively Wallpaper property definitions
```
