## Usage

### Standalone (browser)

Open `index.html` in any modern browser. A settings gear (âš™) in the corner opens the in-page control panel.

### Lively Wallpaper

1. Add `index.html` as a new wallpaper source in [Lively Wallpaper](https://github.com/rocksdanister/lively).
2. Use the bundled `LivelyProperties.json` to expose all settings in Lively's property panel (speed, colors, Conway mode, etc.).
